Media Server API Documentation

This API allows you to manage a collection of movies, TV series, and songs.

Movies

Get All Movies

GET /movies

Response:

[
    {
        "id": 1,
        "title": "The Batman",
        "genre": "Action",
        "director": "Matt Reeves",
        "year": 2022,
        "rating": 7.9
    },
    ...
]

Add a Movie

POST /movies

Request Body (required fields: title, director, year):

{
    "title": "New Movie",
    "director": "Director Name",
    "year": 2023,
    "genre": "Action",
    "rating": 8.5
}

Update a Movie

PUT /movies

Request Body (required field: id):

{
    "id": 1,
    "title": "Updated Title",
    "rating": 9.0
}

Delete a Movie

DELETE /movies

Request Body (required field: id):

{
    "id": 1
}

TV Series

Get All Series

GET /series

Response:

[
    {
        "id": 1,
        "title": "The Last of Us",
        "genre": "Drama",
        "creator": "Craig Mazin",
        "seasons": 1,
        "year": 2023,
        "rating": 8.9
    },
    ...
]

Add a Series

POST /series

Request Body (required fields: title, creator, seasons):

{
    "title": "New Series",
    "creator": "Creator Name",
    "seasons": 1,
    "year": 2023,
    "genre": "Drama",
    "rating": 8.5
}

Update a Series

PUT /series

Request Body (required field: id):

{
    "id": 1,
    "title": "Updated Series Title",
    "seasons": 2
}

Delete a Series

DELETE /series

Request Body (required field: id):

{
    "id": 1
}

Songs

Get All Songs

GET /songs

Response:

[

        "title": "Die For You",
        "artist": "The Weeknd",
        "genre": "R&B",
        "year": 2022,
        "duration": "4:20"
    },
    ...

    "year": 2023,
    "genre": "Pop",
    "duration": "3:30"
}

Update a Song

PUT /songs

Request Body (required field: id):

{
    "id": 1,
    "title": "Updated Song Title",
    "duration": "4:00"
}

Delete a Song

DELETE /songs

Request Body (required field: id):

{
    "id": 1
}
